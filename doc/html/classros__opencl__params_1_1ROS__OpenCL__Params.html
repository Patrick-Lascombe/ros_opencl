bool ros_opencl_params::ROS_OpenCL_Params::two_dimensional

True if the global work size should be two dimensional with the sizes of the two buffers. This is equivalent to using the buffers_size parameter, and pushing back the sizes of the two input vectors. Defaults to false, and is not used in functions with one input vector.
